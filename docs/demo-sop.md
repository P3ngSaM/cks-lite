# CKS Lite 演示 SOP（老板版 / 录屏版）

> 目标：用最短路径展示「老板下发任务 → 数字员工执行 → 结果交付 → 回写闭环」。
> 建议时长：5~8 分钟。

## 0. 演示前准备（1 分钟）

- 启动后端与桌面端，确保页面可正常进入。
- 进入 `老板看板`，确认已选择正确组织（如 `default-org` 或你的演示组织）。
- 进入 `工作台`，确认顶部显示 `AI总指挥` 已开启。
- 进入 `技能管理`，确认已有基础技能列表（至少有 `find-skills`）。

## 1. 技能自动发现与安装（1 分钟）

页面：`技能管理`

1. 在“AI 自动搜索并安装技能”输入框输入目标（示例）：
   - `我要自动整理桌面文件并生成每周总结报告`
2. 选择安装数量（建议 1~2 个）。
3. 点击 `一键自动安装`。
4. 口播重点：
   - “用户不需要先懂 skill 名称，系统先推荐再安装。”
5. 预期结果：
   - 下方日志出现 `已安装：owner/repo` 或明确失败原因。

## 2. 老板派单（1 分钟）

页面：`老板看板`

1. 在“老板派单中心”选择项目、填写任务标题、负责人、执行要求。
2. 勾选：
   - `自动设为下一任务`
   - `派单后立即进入 Workbench`（如果当前页面有该选项）
3. 点击派单按钮。
4. 口播重点：
   - “老板在看板下发任务，不用切多个系统。”
5. 预期结果：
   - 成功提示；
   - 跳转到工作台并绑定任务（顶部出现任务 ID 标签）。

## 3. 总指挥自动执行（2 分钟）

页面：`工作台`

1. 发送任务指令（示例）：
   - `请按任务要求执行，先给我计划，再交付可验证结果。`
2. 展示三个区域：
   - `AI 总指挥执行面板`（理解目标 / 自动编排 / 交付结果）；
   - `结果快照`（可点击文件/链接）；
   - `交付报告`（耗时、成功率、异常）。
3. 点击结果快照中的产出项，直接打开文件或链接。
4. 口播重点：
   - “不是工具罗列，而是可验收交付。”

## 4. 回写闭环（1 分钟）

页面：`工作台`

1. 点击 `回写任务 #ID`。
2. 点击 `导出交付报告`（本地下载 Markdown）。
3. 口播重点：
   - “执行结果自动沉淀到任务日志，可审计、可复盘。”

## 5. 管理视角复盘（1 分钟）

页面：`目标管理` + `老板看板`

1. 在目标管理查看层级：
   - `KPI → OKR → 项目 → 任务`
2. 打开任务详情与审计日志，确认本轮结果可追踪。
3. 回到老板看板看统计卡、负责人状态变化。

## 演示话术（精简版）

- “我们把复杂的 skill 选择交给 AI 总指挥，用户只提目标。”
- “从老板派单到数字员工执行，再到回写审计，形成闭环。”
- “每轮都有可验证产出，不是只输出一段文字。”

---

## 常见故障与兜底话术

- 技能安装失败：
  - 原因多为网络或仓库不可访问；
  - 兜底：切换到已安装技能继续演示。
- 工具执行慢：
  - 网络检索和模型推理存在波动；
  - 兜底：展示交付报告与当前步骤状态，不空等。
- 结果快照打不开文件：
  - 本地权限或路径问题；
  - 兜底：使用“导出交付报告”证明执行链路已产出。

## 验收清单（录屏后打勾）

- [ ] 能从老板看板派单并进入工作台
- [ ] 工作台总指挥可自动执行并展示计划卡
- [ ] 有可点击的交付产出（文件/链接）
- [ ] 能回写任务日志并导出交付报告
- [ ] 目标/看板可看到任务状态更新
