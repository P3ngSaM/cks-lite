# 风险清单（Risk Register）

更新时间：2026-02-06

## R-001 终端策略误拦截合法命令
- **等级**：高
- **现状**：已缓解（已做引号感知 token 解析、大小写安全比较）
- **影响**：桌面自动化可用性下降，用户体验受损
- **措施**：
  - 保留原始脚本路径 token；
  - 仅命令/子命令做大小写无关比较；
  - 持续补充策略回归测试样例（含空格路径、相对路径、大小写路径）

## R-002 终端策略被绕过执行高风险脚本
- **等级**：高
- **现状**：已缓解（限制 python/node 脚本在工作目录内，禁 stdin 脚本模式）
- **影响**：本地文件或环境可能被越权修改
- **措施**：
  - 保持白名单优先策略；
  - 后续将策略迁移到可配置文件并签名校验；
  - 增加安全审计告警（高危命令尝试）

## R-003 启动时自动安装依赖引发供应链风险
- **等级**：高
- **现状**：已缓解（默认关闭，需 `CKS_AUTO_INSTALL_DEPS=1` 显式开启）
- **影响**：启动稳定性与安全性风险
- **措施**：
  - 打包时预装依赖；
  - 线上默认禁用动态安装；
  - 制定依赖版本锁定策略

## R-004 MCP Readiness 假阳性
- **等级**：中
- **现状**：已缓解（仅 200/400 视为可达）
- **影响**：技能状态判断失真，误导用户
- **措施**：
  - 保持 bridge 可达性探测；
  - 后续引入更细粒度健康检查接口

## R-005 审计日志查询性能退化
- **等级**：中
- **现状**：已缓解（改为倒序流式读取）
- **影响**：排障体验下降，接口延迟增大
- **措施**：
  - 后续加索引文件或分段归档；
  - 增加查询耗时监控

## R-006 UI 中英文混杂影响客户演示
- **等级**：中
- **现状**：进行中（Skills/聊天核心已中文化）
- **影响**：客户理解门槛高，影响成交体验
- **措施**：
  - 全站文案统一中文（本周完成）；
  - 下阶段接入 i18n 多语言机制（zh-CN / en-US）

## R-007 关键路径缺少一键回归脚本
- **等级**：中
- **现状**：已启动（新增 `agent-sdk/scripts/sprint1_api_smoke.py`）
- **影响**：改动后回归成本高
- **措施**：
  - 将 smoke 脚本纳入发版前检查；
  - 继续补全对话触发与权限流测试
