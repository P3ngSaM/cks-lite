# CKS Lite 演示脚本（10 分钟）

## 1. 演示目标
- 展示主链闭环：目标创建 -> 执行 -> 回写 -> 人工验收 -> 返工/下一步。

## 2. 演示前检查（2 分钟）
- 启动桌面端，确认可进入 `工作台`、`目标` 页面。
- 可选：先注入一批真实演示数据（推荐）
  - `python scripts/seed_realistic_demo_data.py --reset`
- 在 `目标` 页面点击 `一键生成演示数据`。
- 打开 `任务列表`，确认有 3 条演示任务。

## 3. 闭环演示流程（6 分钟）
1. 在 `目标` 页面选择一条任务，点击 `开始执行`。
2. 跳转到 `工作台` 后，点击 `Plan：生成计划`，等待模型输出。
3. 点击 `Do：开始执行`，展示工具调用和结果回传。
4. 点击 `Verify：验证交付`，展示验收前自检结论。
5. 若任务已回写完成，点击提示卡 `前往任务验收`。
6. 在任务详情抽屉点击 `驳回返工`（或 `验收通过`）。
7. 自动回到 `工作台`，展示结果提示条与快捷动作：
   - 驳回：`一键生成返工计划` / `开始返工第一步`
   - 通过：`规划下一阶段任务`

## 4. 可选展示点（2 分钟）
- `仅看待验收` 快速筛选。
- 多选任务后执行 `批量验收通过`。
- 任务详情中的 `一键回放任务审计日志`。

## 5. 演示常见问题
- 看不到任务：在目标页点击 `应用筛选` 或清空筛选项后重试。
- 无法回写：确认在工作台会话中绑定了 `goal_task_id`（顶部蓝色提示）。
- 页面加载慢：首次懒加载属正常，已做路由拆包优化。
