# Demo Office Assistant

## Purpose
This skill is designed for product demos in Workbench. It provides practical office-style tools:
- build a PPT outline and email draft
- organize desktop files
- summarize a local folder into a document

## Suggested prompts
- "请帮我做一份关于 AI 工作台 的 PPT 并起草发送邮件"
- "请整理我的桌面文件并生成整理报告"
- "请读取桌面文档并输出总结文档"

## Notes
- File operations support `dry_run` for safe preview.
- The skill only handles local filesystem paths supplied by the user.
